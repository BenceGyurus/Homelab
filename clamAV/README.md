# ClamAV Telepítő

Ez a mappa egy szkriptet tartalmaz a [ClamAV](https://www.clamav.net/) és függőségeinek telepítéséhez.

## Mire jó?
- Vírusirtó és kártevő-kereső telepítése Linux alapú rendszerekre.

## Használat
Futtasd a szkriptet root jogosultsággal:
```bash
sudo ./run.sh
```
A szkript telepíti a fejlesztői eszközöket és a ClamAV-hoz szükséges könyvtárakat.
