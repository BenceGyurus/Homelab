# Cleanuparr

Ez a mappa a [Cleanuparr](https://github.com/cleanuparr/cleanuparr) szolgáltatást tartalmazza.

## Mire jó?
- Automatikus takarítás az *arr (Sonarr, Radarr, stb.) szolgáltatásokhoz.

## Használat
Indítsd el a konténert:
```bash
docker compose up -d
```
A konfiguráció a `/home/bence/torrent/data/cleanuparr/config` mappában tárolódik.
Elérhető a 11011-es porton.

## Szolgáltatások
- **cleanuparr**: A takarító szolgáltatás.
