# Maintainerr

Ez a mappa a [Maintainerr](https://github.com/maintainerr/maintainerr) takarító szolgáltatást tartalmazza Plex/Emby/Jellyfin médiatárakhoz.

## Mire jó?
- Médiahia-szabályok alapú automatikus törlése a tárhely felszabadítása érdekében.

## Használat
Indítsd el a konténert:
```bash
docker compose up -d
```
A szolgáltatás a 6246-os porton érhető el.

## Szolgáltatások
- **maintainerr**: A fő alkalmazásszerver.
