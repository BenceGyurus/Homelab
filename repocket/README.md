# Repocket

Ez a mappa a [Repocket](https://repocket.co/) szolgáltatást tartalmazza.

## Mire jó?
- Passzív jövedelem szerzése a nem használt internet-sávszélesség megosztásával.

## Használat
1. Másold le az `.env.example` fájlt `.env` néven:
   ```bash
   cp .env.example .env
   ```
2. Add meg az API kulcsodat az `.env` fájlban.
3. Indítsd el a konténert:
   ```bash
   docker compose up -d
   ```

## Szolgáltatások
- **repocket**: Az ügynök szoftver.
