# Gyürüs Docs

Ez a mappa a saját dokumentációs szervert tartalmazza.

## Mire jó?
- Saját készítésű dokumentációk hosztolása.

## Használat
Indítsd el a konténert:
```bash
docker compose up -d
```
A szolgáltatás a 3010-es porton érhető el.

## Szolgáltatások
- **docs-server**: Egyszerű webszerver a dokumentációkhoz.
