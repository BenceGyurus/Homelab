# Watchtower

Ez a mappa a [Watchtower](https://containrrr.dev/watchtower/) szolgáltatást tartalmazza.

## Mire jó?
- Automatikusan frissíti a futó Docker konténereket, ha új image érhető el.
- Törli a régi, használaton kívüli image-eket.

## Használat
Indítsd el a konténert:
```bash
docker compose up -d
```

## Szolgáltatások
- **watchtower**: A frissítő szolgáltatás.
