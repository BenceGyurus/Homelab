# AdGuard Home

DNS alapú reklámblokkoló.

## Személyre szabás
1. **Hálózat**: Alapértelmezés szerint `host` módban fut, hogy közvetlenül kezelje a DNS kéréseket.
2. **Kötetek**: A `./conf` és `./work` mappákban tárolja a beállításokat.

## Használat
```bash
docker compose up -d
```
A kezdeti beállítás a 3000-es porton érhető el.
