# Crafty Controller

Ez a mappa a [Crafty Controller](https://craftycontrol.com/) Minecraft szerver kezelőt tartalmazza.

## Mire jó?
- Minecraft szerverek futtatása, kezelése és monitorozása webes felületen.

## Használat
Indítsd el a konténert:
```bash
docker compose up -d
```
A vezérlőpult a 8443-as porton (HTTPS) érhető el.

## Szolgáltatások
- **crafty**: A fő vezérlőpult és szervergazda.
